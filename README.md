# Mathématiques pour l'Informatique

Ce dépôt contient l'ensemble des ressources pédagogiques, cours, exercices et projets pour le cours de **Mathématiques pour l'Informatique**.

## Contenu du Cours

Le matériel est organisé autour des thématiques fondamentales suivantes :

### 1. Logique et Raisonnement
*   Introduction à la logique propositionnelle et des prédicats.
*   Méthodes de preuve et raisonnement mathématique.
*   Exercices et quiz d'application.

### 2. Ensembles, Relations et Fonctions
*   Théorie naïve des ensembles.
*   Relations binaires, équivalence et ordre.
*   Injections, surjections, bijections et dénombrement.

### 3. Probabilités
*   Analyse combinatoire.
*   Espace probabiliste et probabilité conditionnelle.
*   Variables aléatoires discrètes et continues.

### 4. Statistiques
*   Statistique descriptive (univariée et bivariée).
*   Visualisation de données (Matplotlib, Seaborn).
*   Introduction à la régression linéaire.

### 5. Algèbre Linéaire et Algorithmique
*   Matrices, vecteurs et systèmes linéaires.
*   Lien avec l'informatique et les algorithmes.

## Outils et Technologies

*   **Python** : Langage principal pour les impémentations.
*   **Jupyter Notebooks** : Support interactif pour les cours et exercices (`.ipynb`).
*   **Bibliothèques** : `numpy`, `pandas`, `matplotlib`, `seaborn`, `statsmodels`.
*   **LaTeX** : Pour la rédaction de documents scientifiques.

## Utilisation

1.  Clonez ce dépôt.
2.  Installez les dépendances Python nécessaires :
    ```bash
    pip install pandas numpy matplotlib seaborn statsmodels
    ```
3.  Ouvrez les notebooks Jupyter pour explorer les concepts mathématiques appliqués.
